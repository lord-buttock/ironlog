## Feature 1 — Previous session notes reminder

**What it does:** When the user is doing an exercise during a workout, if they wrote notes for that same exercise in their last session, show those notes as a small read-only reminder banner.

**Where:** Inside the `phase === 'workout'` block in `ActiveWorkout`. After the coach modification note block (ending ~line 1746), before the `{/* Sets */}` block at line 1748.

**How:** `getLastLogged(sessions, exId)?.notes` is already in scope and returns the previous session's exercise object. Check for a non-empty `notes` string and render the banner if present.

**Code to insert:**

```jsx
{/* Previous session notes reminder */}
{(() => {
  const lastNotes = getLastLogged(sessions, exId)?.notes;
  return lastNotes ? (
    <div style={{ display: 'flex', alignItems: 'flex-start', gap: 10, background: C.dim, border: `1px solid ${C.border}`, borderLeft: `3px solid ${C.muted}`, borderRadius: 10, padding: '10px 14px', marginBottom: 14 }}>
      <span style={{ fontSize: 16, lineHeight: 1.4, flexShrink: 0 }}>📝</span>
      <div style={{ fontSize: 12, color: C.muted, lineHeight: 1.5, fontFamily: C.fMono }}>{lastNotes}</div>
    </div>
  ) : null;
})()}
```

## Feature 2 — Muscle diagram collapsed by default

**What it does:** The `MuscleDiagram` section starts collapsed every time the user arrives at an exercise. A tappable row shows the primary muscle name and a `▼ Show` / `▲ Hide` toggle. Collapses again when navigating to a different exercise.


```js

```

### Step 3 — Replace the muscle diagram block (lines 1690–1697)

**Remove:**
```jsx
{/* Muscle diagram */}

  <MuscleDiagram
```

**Replace with:**

```jsx
  <button
    onClick={() => setShowDiagram(v => !v)}

    <span style={{ fontFamily: C.fMono, fontSize: 11, color: C.muted, marginLeft: 'auto' }}>
      {showDiagram ? '▲ Hide' : '▼ Show'}
    </span>
  </button>

    <div style={{ display: 'flex', justifyContent: 'center', marginTop: 8 }}>
      <MuscleDiagram
        primary={def.primary || []}
        secondary={def.secondary || []}
        size="medium"
      />
    </div>
  )}
</div>
```
